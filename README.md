
# SCSCore1 - Enterprise Core Architecture

## 📌 Описание
**SCSCore1** — это ядро для системы управления серверами с модульной архитектурой.  
Оно обеспечивает надёжную, потокобезопасную и расширяемую платформу для управления узлами, выполнения скриптов, логирования и взаимодействия с оборудованием (Arduino, Web, Telegram и т.д.).

---

## 🧩 Архитектура

Проект построен вокруг **шины событий (Event Bus)** и состоит из следующих ключевых компонентов:

| Компонент | Назначение |
|-----------|------------|
| **AppBus** | Потокобезопасная шина событий. Отправка и получение событий между модулями. |
| **CoreManager** | Управление жизненным циклом модулей: регистрация, запуск, остановка. |
| **ConfigManager** | Загрузка конфигураций из JSON-файла. Доступ к настройкам модулей. |
| **IModule** | Базовый интерфейс для всех модулей. |
| **Core** | Ядро системы. Регистрирует модули, обрабатывает системные события. |

## 🚀 Добавление нового модуля
1. Скопируй папку `Modules/ExampleModule/` → `Modules/NewModule/`
2. Переименуй класс в `NewModule`
3. Реализуй логику в методах:
   - `init()` — инициализация
   - `shutdown()` — остановка
   - `onEvent()` — обработка событий
4. Добавь в `Core::registerAllModules()`:
   ```cpp
   m_manager->registerModule(new NewModule());
   ```
5. Добавь файлы в `CMakeLists.txt`
